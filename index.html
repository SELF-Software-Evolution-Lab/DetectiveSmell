Descripción del proyecto

DetectiveSmell es una extensión innovadora para Visual Studio Code diseñada para mejorar la calidad del código y la productividad del desarrollador mediante un análisis exhaustivo de los code smell. Esta herramienta aprovecha tecnologías avanzadas como ANTLR4, ts-morph y java-ast para proporcionar un análisis detallado de áreas susceptibles a errores y mejoras en el código. Compatible con una amplia variedad de frameworks, incluyendo SpringBoot, Angular y NestJS, DetectiveSmell se adapta a las necesidades de desarrollo de cualquier proyecto. Con características como reglas personalizables, integración sencilla con Visual Studio Code y la capacidad de priorizar esfuerzos de refactorización, DetectiveSmell ayuda a mantener una alta calidad de código y garantizar el éxito a largo plazo del proyecto.
